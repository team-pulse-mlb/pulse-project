# 기능 테스트

> 결과는 검증 완료 시점에 기록한다.

## 1. 기준과 기록 방법

- 기준 문서: `docs/product/FEATURE_SPEC.md`, `docs/product/USER_FLOW.md`
- 테스트 단위: 사용자가 화면에서 수행하는 주요 여정
- 결과 열: 실제 검증 후 `통과`, `실패`, `보류` 중 하나와 필요한 근거를 기록한다.
- 자동화·수동 검증을 수행하지 않은 항목은 결과를 공란으로 유지한다.
