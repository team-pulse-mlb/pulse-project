# PULSE 기획서

## 1. 서비스 개요

**PULSE(펄스)** 는 스포일러 프리 야구 관전 타이밍 추천 서비스다. 경기의 맥박처럼 실시간으로 변하는 흐름을 잡아, 사용자가 지금 볼 만한 경기와 종료 경기의 핵심 흐름을 빠르게 파악하도록 돕는다.

PULSE는 점수 앱이나 영상 하이라이트 서비스가 아니다. 점수와 결과를 먼저 보여주기보다, 스포일러를 줄인 상태에서 관전 결정을 돕는 서비스다.

```text
여러 경기 중 지금 볼 만한 경기 찾기
진행 중 경기의 긴장 구간 놓치지 않기
종료 경기의 긴장 곡선과 이벤트를 결과 노출 없이 확인하기
```

## 2. 해결하는 문제

MLB는 하루에 여러 경기가 동시에 열린다. 경기 시간도 길고 흐름이 느린 구간이 많아, 사용자가 모든 경기를 처음부터 끝까지 보기 어렵다.

기존 스포츠 앱은 점수, 홈런, 역전, 최종 결과를 먼저 알려주는 경우가 많다. 실시간 확인에는 유용하지만, 나중에 보려는 사용자에게는 스포일러가 된다.

PULSE가 해결하는 문제는 다음과 같다.

| 문제 | PULSE의 접근 |
|---|---|
| 어떤 경기를 볼지 고르기 어렵다 | 진행 중 경기의 관전 가치를 정렬해 보여준다. |
| 중요한 순간을 놓치기 쉽다 | 경기 흐름이 높아지는 시점을 알림과 태그로 알려준다. |
| 알림이 스포일러가 된다 | 점수·승패·결과를 제외한 문구로 알린다. |
| 종료 경기 하이라이트가 결말을 노출한다 | 긴장 곡선과 보호용 `경기 흐름`을 제공한다. |

## 3. 주요 사용자

| 사용자 | 필요 |
|---|---|
| 여러 경기를 동시에 보는 야구 팬 | 지금 가장 볼 만한 경기를 빠르게 알고 싶다. |
| 관심 팀 팬 | 경기를 계속 보지 못해도 중요한 흐름은 놓치고 싶지 않다. |
| 스포일러를 피하고 싶은 사용자 | 점수와 승패를 모른 채 나중에 경기를 보고 싶다. |
| 라이트 시청자 | 전체 경기 흐름의 핵심 이벤트를 빠르게 보고 싶다. |

## 4. 차별점

PULSE는 점수 앱이나 영상 하이라이트 서비스가 아니다.

PULSE의 차별점은 스포일러를 줄인 상태에서 관전 결정을 도와준다는 점이다.

| 유형 | 예시 | 한계 | PULSE의 차별점 |
|---|---|---|---|
| 공식 리그 앱 | MLB App, MLB Gameday | 점수와 결과 중심이다. | 점수 없이 볼 만한 순간을 추천한다. |
| 스포츠 스코어 앱 | ESPN, Apple Sports, SofaScore | 실시간 점수 확인에는 좋지만 경기 선택 이유는 약하다. | 여러 경기 중 지금 볼 만한 경기를 정렬한다. |
| 스포일러 방지 서비스 | NoSpoilerz 등 | 주로 종료 경기 보호에 집중한다. | 실시간 경기와 종료 경기를 함께 다룬다. |
| 흥미도 점수 서비스 | Thuuz 등 | 점수는 주지만 야구 맥락 설명이 부족할 수 있다. | 이닝, 주자, 아웃카운트, 투수 흐름을 반영한다. |

## 5. 데이터와 제약

PULSE는 balldontlie MLB API를 사용한다. 예상 플랜은 MLB GOAT Plan이며, 호출 제한은 600 requests/min이다.

| 데이터 | 활용 |
|---|---|
| Games | 경기 목록, 상태, 이닝, 점수 차 계산 |
| Plays | 플레이 흐름, 득점 발생 여부, 볼카운트 |
| Plate Appearances / Pitches | 주자 상황, 강한 타구, 투수 흐름 |
| Lineups | 라인업, 선발 투수, 관심 선수 출전 여부 |
| Season Stats / Standings | 선수와 팀의 시즌 성적, 순위 경쟁 |
| Odds | 경기 전 기대 접전도 계산 |

점수, 승패, 배당처럼 결과 방향을 드러낼 수 있는 정보는 기본 UI에 노출하지 않는다.

## 6. 핵심 기능

### 6.1 실시간 경기 추천

진행 중인 경기의 관전 가치를 계산해 지금 볼 만한 순서로 보여준다. 사용자는 점수나 우세 팀을 보지 않고도 다음과 같은 정보로 판단한다.

```text
후반 긴장 구간
득점권 압박
접전 흐름
관심 팀 경기에서 주목할 흐름
```

로그인 사용자의 관심 팀과 관심 선수는 추천 순서에 가산 요소로 반영한다.

### 6.2 스포일러 보호와 공개

진행 중·종료 경기 상세는 보호 모드로 시작한다. 사용자가 경기 단위로 공개 전환을 선택하면 점수와 상세 결과를 볼 수 있다.

| 경기 상태 | 보호 모드 | 공개 모드 |
|---|---|---|
| 진행 중 | 점수와 승패를 숨기고 이닝 숫자, 현재 상황, 보호 안전 `경기 흐름`을 보여준다. | 점수, 이닝 초·말, 현재 타자·투수, 번역된 최근 플레이를 보여준다. |
| 종료 | 최종 점수와 승패를 숨기고 보호용 AI 헤드라인, 이닝 단위 긴장 곡선, 보호 안전 이벤트를 보여준다. | 공개용 AI 헤드라인, 최종 점수와 승패, 이닝별 점수, 하프이닝 단위 긴장 곡선, 득점 플레이, 번역된 최근 플레이를 보여준다. |

### 6.3 관전 타이밍 알림

경기를 계속 확인하지 않아도 볼 만한 시점을 놓치지 않도록 다음과 같이 안내한다.

| 안내 | 설명 | 전달 방식 |
|---|---|---|
| 급상승 경기 | 관전 가치가 기준 이상으로 빠르게 높아진 경기를 알려준다. | 인앱 토스트와 알림 센터 |
| 관심 팀 경기 시작 | 사용자가 관심 팀으로 등록한 팀의 경기 시작을 알려준다. | 인앱 토스트와 알림 센터 |
| 경기 전환 안내 | 현재 보고 있는 경기보다 관전 가치가 높은 경기가 생기면 이동을 제안한다. | 경기 상세 화면 토스트 |

모든 알림과 전환 안내는 점수, 승패, 우세 팀을 포함하지 않는다. 사용자는 관심 팀과 알림 수신 여부를 설정할 수 있다.
